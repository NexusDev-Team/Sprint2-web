# NEXUS VISION

Projeto desenvolvido para a Sprint 2 de Web Development e Front-End Engineering.

---

## Sobre o Projeto

O Nexus Vision é uma simulação de uma câmera inteligente com Inteligência Artificial integrada, criada para auxiliar usuários na captura de fotos e documentos de forma mais simples, intuitiva e automatizada.

A proposta do projeto é transformar a experiência da câmera mobile utilizando recursos visuais e interativos desenvolvidos com HTML, CSS e JavaScript puro.

---

## Objetivo

Desenvolver uma interface web moderna e responsiva que simule o funcionamento de uma câmera inteligente capaz de:

- auxiliar o usuário em tempo real;
- detectar documentos;
- otimizar capturas;
- organizar imagens automaticamente.

---

## Funcionalidades

### Home
- apresentação do projeto;
- slideshow interativo;
- dark mode;
- menu responsivo;
- mensagem personalizada.

### Câmera Inteligente
- simulação de câmera;
- ativação de IA;
- dicas contextuais;
- modos de captura.

### Scanner
- detecção simulada de documentos;
- exportação em PDF;
- feedback visual.

### Galeria Inteligente
- filtros dinâmicos;
- categorização de imagens;
- organização automática.

### Login
- formulário de autenticação;
- validação de campos;
- mensagens de erro e sucesso.
